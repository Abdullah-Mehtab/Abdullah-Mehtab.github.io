Guestbook

Leave evidence that a human passed through.

This replaces the old localStorage-only comments page with a reusable comment system. In preview mode, comments stay in your browser. With Supabase configured, comments become live-backed, moderated, and reusable under projects, labs, the CV page, and this wall.

How comments work

Local today, backend-ready tomorrow.

The UI already supports Supabase-backed comments with moderation. Add your Supabase URL and anon key in assets/js/site-config.js, run the SQL schema in supabase/schema.sql, and approved comments will be read from the database.

Submissions are pending by default. That keeps spam and nonsense from becoming permanent furniture.

Visitor test

The site should reward attention.

If someone mentions the footer truck, the money line, or the never-ending list, you know they actually looked around. This is analytics, but funnier and less creepy.
